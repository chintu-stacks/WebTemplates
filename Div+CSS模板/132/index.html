pins© is a template design totally formatted in CSS and DIV CONTAINERS. It is was tested to run with Internet Explorer 7, FireFox 2, Opera 9 and Netscape 7/9 on Windows® systems. My special thank's to Irum Shahid (irum on sxc) for the excellent title photo and the permissin to use it here.

 

The menue is a a text construction, easy to modify, that scales downwards. The copyrightbox and the place for a counter adjust themselves automatically and can also easily be changed by inserting text/code, no grafic modifications are needed to run the page. Simply put in your homepage title and maybe adjust it by CSS if you like. You will find a description of what any DIV CONTAINER is good for in the source code.

 

If you insert any text, do it in <p>...</p> elements to have the correct display in all browsers. Especially netscape will cause problems if you don't do so. There are two headline styles to increase the value of your contents to search robots.

 

 

HEADLINE

You should run a monitor resolution of 1024px width as a minimum, to have the best look of the template. At higher resolutions the template will also work correctly and remain readable. There were only standard fonts used in the STYLES CSS, everybody will have installed. So get started and have fun......

 

And at least, the template is written in valid HTML 4.01.
