andreas05 - the simple solution!

Sometimes, the simple things are the most beautiful. This design was made with simplicity in mind, to make it quick and easy to edit the code and get your website online. The amount of code is very small, and the layout is very basic. No visual effects, no advanced features, just a very simple template.

Note: If you don't like the dragon (which was made using a freeware font named "Tattoo"), you can easily replace it with an image of your own!

An open source design!

This design is released as open source, which means that you can make any changes you may want to, and use the page in any way you want to as long as you give the proper credits to the original author. Have fun! If you want more designs to choose from, you can visit my website where you can download more templates by me.
